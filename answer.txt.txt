Cách tạo repo, push code lên repo
1. Tạo repo trên web GitHub
2. Tại terminal, nhập câu lệnh
git init
git commit -m"init project"
git add .
git commit -m"init project"
git remote add origin git@github.com:thanhhalethi/git-course.git
git push origin main

Push code lên git sau mỗi update
git add .
git commit -m "your commit message"
git push origin main

Cách tạo project playwright từ ban đầu
Tạo thư mục trên local
Mở thư mục đó trên terminal
Gõ câu lệnh npm init playwright@latest để khởi tạo playwright
Gõ câu lệnh code . để mở thư mục đó tại trình duyệt code
